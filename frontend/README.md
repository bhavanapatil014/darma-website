# Darma E-commerce Application

A premium, responsive e-commerce website for dermatology products.

## Technologies
- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS / CSS Variables
- **Language**: TypeScript

## How to Run

### 1. Install Dependencies
If you haven't already, install the necessary packages (this is usually done once):
```bash
npm install
```
*Note: If you see peer dependency warnings, you can safely ignore them or run `npm install --force`.*

### 2. Start Development Server
To run the website locally:
```bash
npm run dev
```

### 3. Open in Browser
Once the server is running, open your browser and visit:
[http://localhost:3000](http://localhost:3000)

## Features
- **Home**: Hero section, featured products.
- **Shop**: Product catalog with category filters.
- **Product Details**: Full details view with "Add to Cart".
- **Cart**: Slide-out shopping cart.
- **Checkout**: Simulated checkout flow.
- **Account**: Login/Register simulation.
